**[M5Stack Unit ENV-Pro](https://docs.m5stack.com/zh_CN/unit/ENV%20Pro%20Unit)** 规格参数如下：

---

## 核心传感器

| 项目 | 参数 |
|------|------|
| **传感器型号** | Bosch BME688 |
| **检测类型** | 温度 + 湿度 + 气压 + **VOC 挥发性有机化合物** + **CO₂ 当量** + **IAQ 室内空气质量** |
| **通信接口** | I2C @ **0x77** |

---

## 各传感器参数

| 传感器 | 测量范围 | 精度 |
|--------|----------|------|
| **温度** | -40 ~ +85°C | ±0.5°C (0~65°C) |
| **湿度** | 0 ~ 100% R.H. | ±3% R.H.，响应时间 8s (τ 0-63%) |
| **气压** | 300 ~ 1100 hPa | 噪声 0.12 Pa（≈1.7cm 海拔），温漂 ±1.3 Pa/K |
| **气体/VOC** | — | 标准扫描速度 10.8s |
